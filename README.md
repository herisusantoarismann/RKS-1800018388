# Tentang Aplikasi

Aplikasi merupakan aplikasi admin bagi pengelolaan seminar. Aplikasi ini dikembangkan dengan React JS.

Berikut file/direktori penting aplikasi :

A. Page

* src/Page

B. Assets

* src/assets

C. URL aplikasi

* src/app.js

Aplikasi juga menggunakan beberapa pendukung diantaranya :

 - React Router v5

- Semantic-React v2

 - Sweetalert2 v10

 - Sass v4
